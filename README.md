
# AI-Assisted Waste Sorter

## Overview

This project is a graduation project focused on the mechanical design and system planning of an AI-assisted waste sorting machine.

The system is designed to classify and sort waste materials such as plastic, glass, and metal using a camera-based detection module and a mechanical sorting mechanism.

At the current stage, the project focuses on the SolidWorks CAD design, mechanism layout, technical drawing, and motion/animation study. Physical prototyping, assembly, and testing are planned for the next project phase.

## System Concept

The system is designed to follow this general process:

1. Waste item enters the detection area.
2. A camera-based module identifies the material type.
3. The sorting mechanism rotates or redirects the chute.
4. The item is guided into the correct collection bin.
5. The system resets for the next item.

The design aims to combine mechanical sorting, computer vision, and mechatronics into one integrated waste sorting system.
